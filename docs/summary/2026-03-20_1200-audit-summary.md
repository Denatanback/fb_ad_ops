## Top 10 Issues

| # | Severity | Area | Issue |
|---|----------|------|-------|
| 1 | CRITICAL | security | Real secrets (Google OAuth, Telegram token, admin password) committed in `.env.example` |
| 2 | HIGH | infra | No production Dockerfile -- only dev target exists, no `next build` or standalone output |
| 3 | HIGH | backend | Import processing is fully synchronous inside HTTP requests -- will timeout on large CSVs |
| 4 | HIGH | ui | No dashboard page at `/` -- the main landing page after login doesn't exist |
| 5 | HIGH | security | Google Drive OAuth tokens stored in plaintext in database |
| 6 | HIGH | backend/ui | No pagination on any list query -- will degrade with moderate data volumes |
| 7 | MEDIUM | ui | Admin pages (`/admin/*`) not accessible from sidebar -- no discoverability for admins |
| 8 | MEDIUM | auth | Admin pages have no `requireRole("admin")` -- any authenticated member can access them |
| 9 | MEDIUM | code quality | Duplicated formatters and label maps across import pages |
| 10 | MEDIUM | security | No file size limit on CSV or creative uploads -- OOM risk on large files |

---

## Top 5 Quick Wins

| # | Effort | Impact | Action |
|---|--------|--------|--------|
| 1 | 30 min | Critical | Replace real secrets in `.env.example` with placeholders, rotate compromised credentials |
| 2 | 15 min | High | Add `requireRole("admin")` to all admin page components and server actions |
| 3 | 15 min | Medium | Create a minimal root page at `/` (even a redirect to `/queue`) |
| 4 | 5 min | Low | Pass `session.user.id` as `uploadedById` in manual import upload action |
| 5 | 15 min | Medium | Move shared `formatByteSize`, label maps, and status labels to `src/lib/formatters.ts` |
