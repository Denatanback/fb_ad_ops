# UI Direction

## Theme
- Dark-first theme.
- Light/dark toggle is required.

## Visual tone
- Keep the visual direction Linear-inspired.
- Favor calm contrast, crisp hierarchy, and restrained styling over decorative flourish.

## Data handling
- Use Airtable/Notion-inspired data interaction patterns.
- Prioritize practical tables, filters, detail pages, and operational views.

## Layout
- Use a compact internal-tool layout.
- Favor a sidebar + topbar + data-focused content area.

## Design system principles
- Build simple, reusable primitives for tables, filters, forms, badges, panels, and detail sections.
- Keep spacing, states, and interaction patterns consistent across operational screens.
- Optimize for speed of use by an internal team working with structured data all day.
