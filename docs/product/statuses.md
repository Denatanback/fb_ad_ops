# Statuses

## Primary lifecycle statuses
- queue
- active
- stopped
- scaling

## Tags / labels
- winner
- loser
- top_ctr

## Rule
Statuses describe where the Creative or Launch is in the workflow.
Tags describe an evaluation or notable result.

Examples:
- A creative can be `stopped` and still have tag `top_ctr`
- A creative can be `scaling` and have tag `winner`
